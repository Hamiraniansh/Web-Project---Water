🌊 Master Prompt: 3D Scrollytelling Website on Water

## 🎯 Project Vision
Build a **3D storytelling/scrollytelling website** themed around *Water*, using a **Glassmorphism/Acrylic UI** inspired by Windows 11. The site must feel premium, immersive, and fluid — combining narrative storytelling with interactive 3D visuals, animations, and expressive typography.

---

## 🏗 Roadmap

### Phase 1 — Concept & Planning
- Define narrative chapters: calm ripples → stormy waves → deep underwater → purity/sustainability.  
- Collect inspiration: agpt.co, HHHusher, Apple product pages.  
- Sketch wireframes with glass panels and scroll sections.  
- Select typography pairings (futuristic sans-serif + expressive serif/display).

### Phase 2 — Design System
- Build a **Glassmorphism UI kit** (cards, buttons, overlays).  
- Color palette: blues, teals, gradients, whites.  
- Prototype in Figma before coding.

### Phase 3 — Development Setup
- Framework: Next.js (React).  
- Styling: TailwindCSS or custom CSS with `backdrop-filter`.  
- Animations: GSAP + ScrollTrigger.  
- 3D visuals: Three.js for water surfaces, droplets, underwater lighting.

### Phase 4 — Core Features
- Intro scene: ripple animation + glassmorphic title card.  
- Scroll chapters: each section reveals a new water state.  
- Interactive elements: hover ripples, draggable droplets, refraction effects.  
- Typography animations: liquid-like transitions.

### Phase 5 — Premium Effects
- Particle systems: bubbles, droplets.  
- Dynamic lighting: caustics, refraction.  
- Micro-interactions: ripple buttons, glowing panels.  
- Optional: ambient water sound design.

### Phase 6 — Testing & Optimization
- Optimize Three.js models, lazy-load animations.  
- Ensure smooth scroll on mobile/tablet.  
- Accessibility: readable fonts, ARIA labels.  
- SEO: structured storytelling content.

### Phase 7 — Deployment
- Host on Vercel/Netlify.  
- Add analytics for engagement tracking.  
- Iterate with feedback.

---

## 🎨 Design Brief

**Theme:**  
Water as a narrative medium, expressed through **Glassmorphism UI** with frosted panels, blurred backgrounds, and layered transparency.

**Core Concept:**  
Scroll-driven storytelling journey through water’s states, with seamless liquid transitions.

**Design Language:**  
- Glassmorphism: frosted glass, blur, transparency, depth.  
- Color palette: blues, teals, gradients, whites.  
- Typography: bold futuristic sans-serif (Inter, Neue Haas Grotesk) + expressive serif/display (Playfair Display).  
- Layout: floating cards, overlapping layers, depth effects.

**Animations & Effects:**  
- Scroll animations with GSAP + ScrollTrigger.  
- Three.js water surfaces, droplets, underwater lighting/refraction.  
- Micro-interactions: hover ripples, glowing buttons.  
- Premium effects: dynamic caustics, liquid typography.

**Tech Stack:**  
- React + Next.js  
- TailwindCSS / custom CSS  
- GSAP + ScrollTrigger  
- Three.js  
- Google Fonts / Adobe Fonts  

**Inspirations:**  
- agpt.co → minimal futuristic storytelling  
- HHHusher → immersive motion-heavy design  
- Apple product pages → smooth scrollytelling transitions
